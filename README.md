# C盘深度清理工具

一个功能强大的Windows磁盘清理工具，可以帮助您释放C盘空间。

## 功能特性

- 临时文件清理：扫描并清理系统临时文件
- 浏览器缓存清理：支持Chrome、Edge、Firefox等主流浏览器
- 系统日志清理：清理Windows系统日志和预读取文件
- 回收站清理：清理所有驱动器的回收站
- **软件残留清理：检测并清理已卸载软件的残留文件和目录**
- **无效快捷方式清理：清理桌面和开始菜单中的无效快捷方式**
- 图形化界面：简洁易用的GUI界面
- 安全确认：清理前需要用户确认，防止误操作
- 实时统计：显示可释放空间和清理进度

## 安装说明

1. 确保已安装Python 3.6或更高版本
2. 安装依赖库：
```bash
pip install -r requirements.txt
```

## 使用方法

1. 运行程序：
```bash
python main.py
```

2. 点击"扫描C盘"按钮开始扫描
3. 查看扫描结果，可以点击"选择"列来选择或取消选择要清理的项目
4. 使用"全选"或"取消全选"按钮快速选择
5. 点击"清理选中"按钮开始清理
6. 在弹出的确认对话框中点击"是"确认清理
7. 查看清理结果和释放的空间

- 用户临时文件夹
- 系统临时文件夹

### 浏览器缓存
- Chrome浏览器缓存
- Edge浏览器缓存
- Edge代码缓存
- Edge GPU缓存

- Edge WebView旧版本
- Edge Core旧版本
- Edge浏览器缓存

- Edge GPU缓存

- 剪映如下缓存内容（贴纸、特效等素材）
- 剪映音频缓存（音频素材临时文件）
- 剪映工作平台缓存（项目工程、预览文件等）
- 剪映艺术特效缓存（特效、滤镜、转场等视觉效果）
- 剪映临时文件

- Windows INetCache
- Windows INetCookies

### 回收站
- 所有驱动器的回收站内容

### 软件残留
- Program Files和Program Files (x86)中的残留目录
- AppData中的软件残留配置文件

### 无效快捷方式

- 开始菜单中的无效快捷方式
- 公共开始菜单中的无效快捷方式

## 注意事项

1. 清理操作不可恢复，请谨慎选择要清理的项目
2. 建议在清理前关闭所有正在运行的程序
3. 某些文件可能正在被系统使用，无法清理
4. 清理后可能需要重启计算机才能完全释放空间
5. 首次使用建议先扫描，了解可清理的内容
6. **清理剪映缓存后，贴纸、特效等素材可能需要重新下载**

## 系统要求

- 操作系统：Windows 7/8/10/11
- Python版本：3.6或更高版本
- 内存：建议512MB以上
- 磁盘空间：至少10MB可用空间

## 故障排除

### 扫描失败
- 确保以管理员身份运行程序
- 检查是否有足够的权限访问系统文件夹

### 清理失败
- 某些文件可能正在被使用，无法删除
- 尝试关闭所有程序后重新清理

### 程序无法启动
- 确保已安装Python
- 检查是否安装了所有依赖库
